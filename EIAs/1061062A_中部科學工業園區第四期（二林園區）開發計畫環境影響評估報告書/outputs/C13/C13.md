# 第十三章 **結論及建議**

一、中科四期(二林園區)**開發計畫之引進產業,於園區轉型後將限定於「低**
用水、低排放」性質,與原一階環說書所載內容相較,用水量由 16 萬 CMD 減為 2 萬 CMD**、需用電量由** 151 萬 kW 減為 38.9 萬 kW、廢 (污)**水量也由** 12 萬 CMD 減為 2 萬 CMD**,進駐產業性質比一般科學**
園區單純,整體能源消耗及污染排放也大幅減少。

二、本計畫積極配合水資源再生政策,園區設立水資源中心,污廢水經妥 善處理後,將依據「再生水資源發展條例」及相關子法規定,提供做 為各項用途之再生水源,力求水資源再生利用最適化(**詳第五章開發** 內容說明)**。除園區進行再利用之外,若區外有意取用之鄰近地區廠**
商或再生水經營業經綜合評估後符合資格者,亦會納入供水考量,同 時亦可配合相關主管機關之需求,提供抑制揚塵等環境友善用途所需 水源。

三、本園區水資源中心之再生水再生利用,不直接排放於地面水體,不僅 可達到水資源循環再利用目標,亦可大幅減少環境污染負荷。

四、本園區營運期間之空氣污染物排放,主要受關切項目為細懸浮微粒
(PM2.5)及 VOCs(衍生臭氧)**。考量中部空品區** PM2.5 現況不佳(已超標)
之情形,本園區將先採行源頭管控儘量減少排放量,同時為進一步 達到環境友善之目的,規劃推動「移動污染源自主管理計畫」,透過 園區運輸活動使用之柴油運輸交通工具排氣污染管制,做為環境友善 措施。另 VOCs **方面,同樣採行源頭管控並承諾排放量≦**200 噸/年, 並配合彰化縣 VOC **排放減量趨勢,選取彰化縣境內固定污染源之** VOC 減量達 220 公噸/**年,做為抵換分析之設定來源,其增量抵換模**
擬結果顯示抵換成效符合規定。

五、本園區預估引進直接就業人口 25,500 **人,彰化縣因就業人口之進駐**
而衍生之居住人口增量約為 19,500 人,在本園區帶來大量高科技人 才進駐後,對人口成長及人力資源的提升有相當正面之助益,同時 可促進地方經濟發展。另本園區營運後將提供更完善且高品質之生 活、交通及遊憩等諸多機能,對當地生活品質有直接的正面效益。

六、本園區若不予開發,雖可避免開發營運行為所產生約 0.11 億元/年之 環境外部成本,惟園區開發後所可能增加的平均約15.37億元/年淨效 益相較,在確實執行環保措施、將環境影響控制在可接受程度的前 提下,本園區計畫具有明顯的開發效益。

七、本園區主方案與其他替代方案(**開發地點、開發方式及環保措施等**)相 比,相關效益及環境友善程度均遠優於各項替代方案。而未來若開 發主方案無法順利推動,而須將某項替代方案轉為主方案時,屆時 亦須依規定辦理環評變更,審查通過後才能執行。

八、本計畫之開發對周圍環境將產生不同程度之影響,惟經採用相關減 輕對策後,環境影響程度將可有效降低,有關各項環境因子之影響 評估及採行對策詳如第十五章摘要表。

九、開發單位將依據環評審查意見及結論,納入工程細部設計詳予考 量,並承諾以降低環境影響為原則進行開發;此外,將落實相關環 保減輕對策及環境監測計畫、緊急應變計畫,以有效管控本開發案 所可能造成之環境影響。
