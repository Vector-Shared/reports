# 地層下陷風險摘要 — 二林園區

來源：彙整自報告第7章（C07）之地質、液化與地盤評估段落及施工建議。

一、結論要點
- 基地地形平坦、坡度小，短中期廣域性地層下陷風險低（在無大量地下水抽取或重大地層改變的前提下）。
- 區域含有潛在液化敏感層，於設計地震或較大震動情況下，局部淺層可能出現液化或瞬時沉陷；此外，軟土壓密或不當回填亦會造成長期沉降。
- 若採行報告建議之地盤改良、適當回填與地下水管理，可將地層下陷風險降至可接受水準。

二、關鍵風險機制
1. 地震液化 — 飽和砂層在震動時孔隙水壓升高導致承載力下降，可能發生瞬時沉陷或地面變形。
2. 長期壓密沉降 — 填土或新負載造成軟土孔隙壓力釋放與壓實，產生持續沉降。
3. 地下水抽取誘發沉降 — 大量抽水會改變孔隙水壓並引起土體壓縮。
4. 施工不良（未壓實、回填方式不當）引起局部不均勻沉降。

三、報告（C07）中已有之要點
- 液化評估：中小震僅局部淺層有輕微液化可能；設計/大地震下潛勢上升。
- 建議對重要設施採用深基礎或局部地盤改良；回填採分層壓實或預壓/真空加固以降低後期沉降。

四、立即可採行之工程與管理措施
- 施工/設計階段：進行詳細土壤、原位試驗與沉降/液化預估；對重要構築採樁基或地盤改良。
- 地下水管理：限制抽水並監控地下水位。
- 監測：設置沉降水準點／沉降板、地下水監測井、裂縫觀測點；施工期頻繁監測（每週或每月），營運期定期（每季或半年）。
- 應變機制：建立門檻值（如超過預估沉降速率或累積沉降達特定值）並啟動調查與加固程序。
