# 生態影響摘要（中部科學工業園區第四期 — 二林園區）

來源
- 主要依據：outputs/C07/auto/C07.md（第7章）及 outputs/C03/C03.md（專家與領域簽名與調查單位），並彙整 EIAs 下 C* 章節調查結果。

摘要結論（要點）
1. 生態整體評估結論：
   - 本案開發範圍以平坦農業與開發用地為主，原始自然棲地有限；生態調查顯示區域內有一般常見之陸域與水生生物群，但未發現大量或極高保育價值之聚落集中。總體生態影響在採行設計上之避讓與緩解措施後，可控制於可接受範圍。

2. 陸域生態（物種與棲地）：
   - 調查發現之植群、昆蟲與鳥類屬於常見種類，無大面積原始林或珍貴濕地被直接破壞。建議保存既有零星綠地、設置生態廊道與緩衝帶，並在景觀設計中採用在地原生植栽以促進生態復育。

3. 水生生態與沿海生態：
   - 近海/河流水域之生態監查包括水生動物與漁業資源調查；簽名名單顯示已針對白海豚等保育物種進行專家評估。評估結論：若施工與排放採取嚴格控管（沉澱、污水處理、施工期擾動最小化），對沿海及河口生態之長期影響可被降至低程度；短期擾動需以監測與施工時段控管降低。

4. 受保育物種與敏感棲地：
   - 報告提及中華白海豚相關調查（專家參與），已評估區域潛在影響；若發現保育類動物活動熱區，應在設計上排除或設定更大緩衝距離，並於施工期採取專門保育措施與監測計畫。

5. 主要生態影響項（關鍵風險）與建議緩解措施：
   - 棲地喪失/破碎化：保留與連接現有碎片化綠地，於重要移動廊道設置植栽與穿越設施。  
   - 施工期侵擾（噪音、沉降、泥沙徑流）：實施施工時降塵、沉澱設施、並限制在敏感期(繁殖季)的大型作業；河川工程採階段性作業並設置防護堤與沉砂池。  
   - 污染物與水質影響（營運期）：確保所有污水經中央化處理並達標排放；對於可能影響沿海/河口生物的排水，使用多段處理與監測。  
   - 監測與回饋：建立生態監測計畫（物種名錄、指標物種、季節性調查）與門檻觸發應變（例：某保育種出現→暫停作業並通報）。

6. 建議監測計畫（概要）
- 施工前基線：完成季節性生態基線調查（鳥類、兩棲爬蟲、重要植群、水生生物）。
- 施工期監測：周邊水質（每週）、沉澱與擴散監測、敏感物種（每月或生殖季）巡查。  
- 營運期長期監測：每季水生/陸域生態監測報告（連續 3–5 年），監測結果納入管理措施調整。 

7. 交流與補償建議
- 若無法避免的重大生態影響，提出生態補償方案（如濕地重建、棲地改善或其他等值補償），並以第三方監測驗證補償成效。
