# 四、 土壤液化分析  

土壤液化係指飽和之砂性土壤，於地震力之反覆剪力作用，使土壤內部孔隙水壓力逐漸增高，導致土體內之有效應力逐漸減小，當有效應力降低至趨近於零時，土壤將喪失大部份或全部的強度而呈液態化之現象。土壤液化發生時，土壤承載力將隨之驟減，結構物將產生明顯之沉陷、傾覆或破壞。  

依據建築耐震設計規範地表下 $10\mathrm{m}$ 以內地下水位才有發生液化之潛能， ${\mathrm{,}}{\mathrm{PL}}{\geq}15$ 為高潛勢區； ${5\le\mathsf{P L}}{<}15$ 為中潛勢區；PL $<5$ 為低潛勢區，本計畫BH-2 經分析後中小地震 ${\mathrm{PL}}{=}2.06$ ，設計地震 $\mathrm{PL}{=}0.49$ ，最大考量地震 $\mathrm{PL}{=}0.42$ ，皆為低潛勢區域，其他各孔位表土層厚度極小，皆屬卵礫石及岩層，故應無液化之可能。

本計畫以最保守狀況推估施工尖峰期間施工人員估計最多約 900 人，人員每人每日用水量為 30 公升，故總用水量每日 27 CMD，生活污水量以用水量 $90\%$ 估算，則施工期間施工人員產生之污水量約為 24.3 CMD。施工人員進駐而產生之生活污水量均透過流動廁所或套裝式污水處理設備妥善收集處理，因此預估施工期間對水體之水質影響應屬輕微。

營建工程因整地開挖所致之土質疏鬆及施工車輛挾帶之土砂，若遇雨水沖刷往往會造成水中懸浮固體物增加；施工人員之生活污水、施工機具、車輛保養清洗與工程廢液亦可能造成污染，因此本開發計畫將於施工階段於廠址四周設置截水溝，基礎施工產生之泥水或地表逕流循截水溝進入沉砂池，使其去除砂土及懸浮固體後始得放流。廠址產生之地表逕流或泥水經妥善處理後，使其水質符合環保法規之營建工地放流水標準（ $\mathrm{{BOD}\{30\ m g/L}}$ ，CO $\mathrm{>100mg/L}\mathrm{~,~}\mathrm{SS}30\mathrm{mg/L}$ ，真色色度 550）。

# 7.1.3 空氣品質  

# 一、 施工期間  

施工期間對於空氣品質之主要影響來自計畫場址工區內整地、電塔基礎工程開挖作業揚塵逸散、施工機具與工程車輛廢氣排放，以及工區外運輸車輛廢氣排放。本案將利用高斯擴散模式「AERMOD」進行工區內整地、電塔基礎開挖作業揚塵、施工機具與工程車輛廢氣排放及工區外運輸車輛廢氣排對空氣品質之影響評估。將針對空氣污染物排放量進行最大濃度增量模擬，並加成背景濃度後與空氣品質標準比較，以評估對於空氣品質可能影響。  

# (一) 施工空氣污染物排放量推估  

# 1. 工區施工作業粒狀物揚塵逸散  

施工期間工區粒狀空氣污染物揚塵逸散主要來自計畫廠址內整地工程以及電塔基礎工程開挖。其影響範圍多侷限於工區內及其附近數十至數百公尺範圍內。  

針對計畫廠址內整地作業之粒狀物逸散量推估，依據目前規劃，未來全區將採分期施工，各期工程最大可能裸露面積以 2 公頃計。綜合參考環境部公告TEDS 12.0 版面源排放量推估手冊之表 $\mathbf{B}2{-}2\mathbf{\Omega}^{\Gamma}$ 「面源逸散性粒狀污染物（B2）排放係數及資料引用來源 $\lrcorner$ ，建築工程之 RC 結構施工項目所產生之總懸浮微粒排放係數 0.200 公斤/平方公尺/月（ $7.72\times10^{-5}$ 公克/平方公尺/秒）。同時，工區裸露面在採灑水作為揚塵防制措施下，粒狀污染物面源排放係數在減量 $60\%$ 後可降為 $3.09\times10^{-5}$ 公克/平方公尺/秒。 $\mathrm{PM_{10}/T S P}$ 及 $\mathrm{PM}_{2.5}/\mathrm{PM}_{10}$ 排放率比例分別為 $0.555\cdot0.1\$ 。  

針對計畫廠址內基礎開挖作業之粒狀物逸散量推估，依據目前規劃，其施工作業面預估為 900 平方公尺。參考環境部公告 TEDS 12.0 版面源排放量推估手冊之表 ${\bf B}2–2^{\Gamma}$ 「面源逸散性粒狀污染物（B2）排放係數及資料引用來源」，管線開挖工程施工所產生之總懸浮微粒排放係數 0.256 公斤/平方公尺/月（ $9.88\times10^{-5}$ 公克/平方公尺/秒）。同時，作業裸露面在採灑水作為揚塵防制措施下，粒狀污染物面源排放係數在減量 $60\%$ 後可降為 $3.95\times10^{-5}$ 公克/平方公尺/秒。 $\mathrm{PM_{10}/T S P}$ 及 $\mathrm{PM}_{2.5}/\mathrm{PM}_{10}$ 排放率比例同上分別為 $0.555\cdot0.1\$ 。  

上述兩項施工作業可能工期重疊，因此本計畫評估時，將保守考慮兩者同時排放並與下列施工機具排放量合計進行加成模擬評估。施工時間預計在8:00${\sim}12{:}00\cdot13{:}00{\sim}17{:}00$ ，共8 小時。

# (二) 營運期間廢熱擴散增溫模擬結果及影響分析  

本計畫未來發電機組的冷卻系統將採用氣冷式，其冷卻方式為將汽輪機出口之蒸汽透過大型管路導引至氣冷式冷凝器之熱交換管，使蒸汽在熱交換管中與空氣作熱交換，因此將會有熱交換後之空氣向外排出。而當大氣中持續有大量熱源進入時，可能會造成局部區域增溫情形。因此，以下將針對新增機組營運期間廢熱排放後可能造成之增溫影響進行擴散模擬評估分析。  

# 1. 模擬條件設定  

本計畫將採用 AERMOD 高斯擴散模式進行熱擴散模擬。配合環境部空氣品質模式支援中心所提供氣象資料，模擬期程選定目前最新民國 111 年全年。本計畫廢熱排放特性，包含冷凝器熱排放以及煙囪熱排放，模擬範圍與營運期間原生性空氣污染物模擬範圍同為 10 公里 $\mathbf{\nabla}\times10\mathbf{\Omega}$ 公里，受體點網格間距同樣設定為100 公尺，其餘參數設定亦悉依原生性空氣污染物模擬之設定。

# (三) 營運期間衍生性空氣污染物模式模擬及空氣品質影響分析  

# 1. 模式模擬執行方式  

本案以網格氣象模式 WRF（4.3.1 版）及網格空氣品質模式 CMAQ（5.2版）進行本計畫營運後臭氧 $(\mathsf{O}_{3})\cdot$ 懸浮微粒（ $\mathbf{\dot{PM}}_{10}\mathbf{\dot{\Phi}},$ ）及 $\sharp_{\sharp}$ 懸浮微粒（PM2.5）濃度最大增量之模擬，並針對模擬結果進行空氣品質之影響評估。  

# 2. 模擬程序及條件設定  

模擬作業程序係依據環境部公告「空氣品質模式模擬規範」之相關規定辦理。CMAQ 模式使用巢狀網格套疊方式進行模擬，共採用 4 層巢狀網格，其中第 4 層採用該模擬規範所指定之臺灣本島作為模擬範圍。  

配合環境部 TEDS 11.1 排放量資料庫基準年 108 年，本案模擬期程係選取環境部空氣品質模式支援中心網站所公告雲嘉地區第一優先案例月，$\mathrm{O}_{3}$ 案例月 108 年 10 月、PM（ $\mathrm{PM}_{10}\cdot\mathrm{PM}_{2.5}\cdot$ ）案例月-108 年 1 月，分別進行 ${\bf O}_{3}\cdot\mathrm{P}{\bf M}_{10}\cdot\mathrm{P}{\bf M}_{2.5}$ 濃度之最大增量模擬。  

完整之網格模式模擬範圍、網格設定、背景排放源處理、氣象資料之產生、邊界及初始條件設定等內容，詳參附錄十一_A11.2.2 節。  

# 3. 模式性能評估  

有關網格模式模擬準確度之檢核，係依照「空氣品質模式模擬規範」相關規定進行模式性能評估。其執行步驟，係根據模擬範圍內背景氣象場資料及背景排放源空氣污染物排放資料（使用環境部公告 TEDS 11.1），針對上述選定之模擬案例月，以 WRF 網格氣象模式模擬案例月期間背景氣象場後，再以 CMAQ 網格空品模式模擬案例月期間 $\mathrm{O}_{3}$ 及 $\mathrm{PM}(\mathrm{PM}_{10}{\cdot}\mathrm{PM}_{2.5})$ 背景污染物濃度場。並分別針對 WRF 模式及 CMAQ 模式的模擬結果，與環境部公告之空氣品質性能評估分區表（本案位於雲嘉地區）所列之中央氣象局各測站氣象觀測結果及環境部空品測站空氣品質觀測結果進行誤差比對，檢視是否符合「空氣品質模式模擬規範」所訂定之容許誤差範圍，以確保本案 O3 及 PM（ $\mathbf{PM}_{10}\cdot\mathbf{PM}_{2.5})$ ）模擬時所使用之氣象場資料具有一定程度之準確性，進而確保其後續空氣品質模擬結果之合理性。  

經背景模擬結果驗證後，本案所使用之網格氣象模式 WRF 及網格空品模式 CMAQ 皆可通過性能評估檢核，因此確認模式具有一定程度之準確性，其後續應用於本案之模擬結果應可被接受。相關模式性能評估之內容說明，詳參附錄十一_A11.2.2 節之十及十一。

# 5. 最大濃度增量之計算方式  

最大濃度增量模擬程序同樣依照環境部「空氣品質模式模擬規範」要求，將本案新增污染源空氣污染物（NOx 及 $\mathrm{NH}_{3}$ ）排放量併入背景排放源排放量清單（即環境部TEDS 11.1）後，針對 $\mathrm{O}_{3}$ 濃度增量模擬部分，以 CMAQ模式模擬出背景加成本計畫新增空污排放後之 $\mathrm{O}_{3}$ 濃度場，再依時間配對、空間配對的方式減去同樣經過 CMAQ 模式模擬之背景濃度場（即「無本計畫空污排放」之情境）後，得出 $\mathrm{O}_{3}$ 小時值濃度、八小時值濃度之最大增量。至於 PM（ $\mathbf{PM}_{10}$ 及 $\mathbf{PM}_{2.5}$ ）濃度增量模擬部分，則採類似操作程序，可得到 $\mathbf{PM}_{10}$ 日平均值濃度、年平均值濃度，以及 PM2.5 日平均值濃度、年平均值濃度增量。最終將以最大增量濃度做為判定影響大小之評估值。有關最大增量濃度計算方式，詳參附錄十一_A11.2.2 節之十二。

# 三、 溫室氣體  

目前火力發電仍然是基載電力最主要的供電來源，火力電廠最主要的溫室氣體排放來自化石燃料燃燒後產生的二氧化碳，因此燃料含碳越低或機組效率越高，所產生的溫室氣體越少。天然氣是目前排放強度最低的化石燃料，因為其低碳及低污染的特性，世界各國均將高效率燃氣機組作為國家溫室氣體減量的重要方法之一，CDM（聯合國清潔發展機制）所提出的溫室氣體減量方法論也包含了天然氣併網發電（方法論編號AM0029），因此提高天然氣發電比例也是我國低碳能源政策中非常重要的一環。

# (三) 本計畫溫室氣體減量友善措施

# 7.1.4 噪音振動  

本計畫評估電廠施工及營運期間對周圍敏感點之噪音及振動之加成影響。以下將分別就噪音、振動兩項目，評估個別於施工期間及營運期間之環境影響。  

# 一、 噪音  

# (一) 評估基準  

本評估工作採用德國 Braunstein+Berndt GMBH 公司所發展之「SoundPLAN」噪音電腦模式進行預測與分析。該模式之特點在於可同時或分別考慮點源、線源及面源等不同型式噪音源及其合成之音量，除可推估個別敏感點之噪音量外，亦可預測整個計畫區內外之等噪音線，將此預測音量與各受體背景音量合成後，再依據環境部建議之噪音影響評估流程圖（圖7.1.4-1）判定影響程度。  

# (二) 施工階段影響預測  

# 1. 施工期間計畫廠區施工  

本計畫工程施工包括整地工程、輸電線工程…等，施工機具數量、種類及噪音量詳表7.1.4-1，鄰近計畫區周界旁最大噪音值約為 65.2 dB(A)，符合第三類營建工程噪音管制日間標準值 72 dB(A)。施工機具噪音預測各施工階段中同時施工機具之合成音量，其中計畫區與東側敏感點受體大崎腳聚落距離約 460 公尺；與北側敏感點受體看松山國小距離約 950 公尺；與北側敏感點受體海恩淨寺距離約 800 公尺；與西南側敏感點受體覆鼎金聚落約1,000 公尺；與東南側敏感點受體佑民祠約510 公尺；與西側敏感點受體計畫廠區約120 公尺。

# 7.1.7 電磁場  

# 一、 評估方法  

依環境部公告「限制時變電場、磁場及電磁場曝露指引」，對本計畫廠址至變電所間架空線/管路路徑附近敏感點進行電磁場檢討計算，並預估完工送電後路徑上電磁場值最大值。  

(一) 計算架空線/管路沿線之敏感點之電磁場最大值，並以本計畫發電計畫之最大總裝置容量，預估架空線/管路沿線敏感點最高之電磁場大小。  
(二) 計算三維空間電磁場源：完整包括水平與垂直走向之 345kV 導線。  
(三) 計算考量：計算時各方向導線均有考慮，分別對導線各相序排列方式進行運算。  
(四) 計算結果：考量 $345\mathrm{kV}$ 導線回線數及電流量，計算結果架空線/管路沿線敏感點最大磁場預估值，均遠低於環境部 $833\mathrm{mG}$ 的參考位準值，最大電場預估值均遠低於環境部 $4167\mathrm{V/m}$ 參考位準值。  
(五) 本計畫架空線部分包括電場與磁場，預估值需考量背景值與本計畫完工送電後之新增計算值。  
(六) 管路部分因電力設備與電力電纜均有金屬層屏蔽，因此對空間中電場的新增值為零，電場預估值即為背景值。磁場預估值需考量背景值與本計畫完工送電後之新增計算值。  

電力頻率磁場計算方法，從相關電磁場理論可推導，電力頻率場源所產生的磁場行為類似靜磁場。本檢討計算利用有限元素法套裝軟體，針對導線三維空間配置及各導線之電流條件模擬演算。

# 7.2 生態環境  

# 7.2.1  陸域生態  

# 一、 陸域植物生態  

# (一) 施工期間  

本計畫施工區域分成電廠及輸電線路兩部分。電廠環境主要為果園及人工林。輸電線路位置主要穿過次生林、人工林、公園綠地、耕地、果園及道路等環境。整體而言，電廠及輸電線路的施工區域從自然度 0 至4。  

施工期間對工程範圍內的植物所產生的影響包括：  

（1）部份工區上的植物遭移除，以致族群減少。如電廠內受工程影響胸徑大於 $10\ \mathrm{cm}$ 的喬木，主要有大葉桃花心木約 4,500 棵、赤桉約 3,150 棵、相思樹約 1,350 棵、白雞油約 400 棵、香楠約 100 棵、黑板樹約 50 棵、茄苳約 20 棵、無患子約 20 棵、耳莢相思樹約 20 棵、銀合歡約 20 棵、樟樹約20 棵、楝約10 棵、厚殼樹約 10 棵、山黃麻約10 棵、白匏子約10 棵、血桐約 10 棵、構樹約10 棵、土密樹約 5 棵、蟲屎約 5 棵、華盛頓椰子約 2 棵及盾柱木約1 棵等等。  

（2）工程引發揚塵覆蓋葉表，可能導致植物生長不佳。  

調查範圍所記錄之稀有植物中，蘭嶼羅漢松距離電廠施工範圍較近，距離電廠位置約 $17\mathrm{m}\cdot$ 為人為栽植的稀有植物，種植於嘉惠電廠側門做為景觀植栽，應不至於移除，但距離施工位置較近，工程引發之揚塵可能覆蓋葉表，植物生長不佳；其餘稀有植物距離電廠及陸纜皆有 $300\mathrm{m}$ 以上。  

# (二) 營運期間  

營運期間因施工結束，輸電線路沿線回復原有地面樣貌，電廠及陸纜施工區人員活動量減少，僅設備定期檢查及維修時有人員活動，故對周遭植被影響較施工期間減少。  

# 二、 陸域動物生態  

# (一) 施工期間  

本計畫為電廠新建開發，而施工期間的人為活動增加、機具的聲響以及振動等干擾因素，將會對周遭野生動物造成驅離的作用，導致其遷移到鄰近的相似環境中，因而導致鄰近相似環境的動物族群量變多，既增加相對競爭壓力，也增加覓食的困難度，尤其對具領域性或保育類物種之影響更大，如具領域性物種因領域受到入侵而產生競爭，保育類則因食物來源及棲地選擇相似而產生競爭。  

由於工程進行時會有噪音振動、工程車輛、人員活動及廢棄物等干擾，可能影響原棲息與活動於工程區內的陸域動物因噪音振動而受驚嚇或驅離，以下針對上述幾點進行說明︰  

# 1. 噪音振動  

施工期間機具的聲響及振動對工程內及周邊的野生動物會造成驅離的作用，導致其遷移到鄰近的相似環境中，因而使該區域內野生動物族群量降低，鄰近相似環境的動物族群量則變得較為豐富，增加相對競爭壓力，然對於具領域性或保育類物種之影響較大，具領域性物種因領域受到入侵而遷移或產生競爭，保育類則因食物來源及棲地選擇相似而產生競爭。  

# 2. 夜間照明  

在燈光方面，夜間工程進行產生夜間照明將增加夜行性物種（如領角鴞）被車輾斃及被人類撞見與捕捉的機會。  

# 3. 工程車輛  

工程車輛的增加會提高工程區及施工便道之野生動物遭車輛撞擊的機會，其中以哺乳類（如食蟹獴）、兩生類、爬蟲類、地棲性及不擅長飛行鳥類（如藍腹鷴）等移動能力較差的動物較易受傷害；施工所帶來的工程車輛及施工人員車輛，將使得既有道路的使用率增加，對於棲息於該區環境內的野生動物將有所影響。  

# 4. 人員活動  

施工作業會增加工程區及鄰近人員活動量，提高動物遭騷擾或獵捕之機會。  

# 5. 廢棄物及化學物質  

施工產生的工程廢棄物、土方及施工人員所產生的民生廢棄物若未妥善集中整理，除會造成環境髒亂外，亦會吸引野生動物翻尋覓食，造成誤食而影響其健康，也容易使活動的人員或動物遭銳物誤傷。此外若該區域施用化學性殺蟲劑、毒鼠藥或除草劑等，將可能導致野生動物（如食蟹獴、領角鴞、黑翅鳶、鳳頭蒼鷹、大冠鷲及東方蜂鷹等）的食物（如鼠類、小型鳥類、蜥蜴或昆蟲等）來源遭受毒物污染。  

# 6. 保育類動物  

調查中記錄黑翅鳶、鳳頭蒼鷹、大冠鷲、東方蜂鷹、藍腹鷴、領角鴞、八哥及臺灣畫眉等 8 種為珍貴稀有保育類野生動物，食蟹獴及紅尾伯勞 2 種

# (二) 營運期間  

1. 人為活動頻度及交通工具增加，伴隨著產生之噪音及光照等因素，可能驚擾到原有棲息於此地之鳥類及動物，造成生活模式改變及覓食空間壓縮。

# 7.2.2 水域生態  

# 一、 施工期間  

本計畫基地範圍位於牛稠溪附近，施工產生的工程泥廢水、施工人員產生之生活廢水、施工機具之廢油（如.機油、汽油、材油及潤滑油等）、施工材料及垃圾囤積物等，若因人為管理不當，而直、間接流入附近水域，將會對本區水質造成污染。  

# 二、 營運期間  

營運期間之廢棄物堆置及民生廢水處理等，若直接排放或棄置臨近水域，將對周邊水域生態將會造成直接（如直接將廢棄物丟棄在水域環境中）及間接（如雨季，透過排水流入鄰近水域生態）之影響。

# 7.3.2 遊憩環境  

# 一、 開發行為影響預測  

為了遊憩品質影響量化評估需要，篩選計畫區與鄰近環境中敏感或較具有代表性之遊憩據點，藉以比較計畫開發前後與執行中可能產生之遊憩品質衝擊，據點之選擇以敏感度較高之地點為代表，並調查與本計畫開發的關聯性。計畫基地多屬台糖公司用地，目前多種植果園及闊葉林，周邊以電廠、工業區、住宅聚落、蔗田果園及雜林地為主，並鄰近交通動線，當地居民、道路使用者及工作者容易注意到本計畫開發行為，未來完工後，大面積的建物量體及相關設施將改變原有視覺印象及既有天際線，可能影響遊程中之視覺體驗，對於遊憩品質將造成不同程度的影響。以下分為施工期間與完工營運期間部分來討論。  

# (一) 施工期間的遊憩影響預測  

# 1. 施工影響遊憩據點的交通可及性  

本工程計畫施工階段，將有施工機具及載運材料卡車進出，其所衍生之交通量將可能對鄰近道路造成可及性之影響，應避免上下班及連續假日施工道路的使用，並研擬施工道路計畫以減低施工期間之交通衝擊。  

# 2. 施工影響鄰近遊憩據點的遊憩體驗  

材料堆置或整地後裸露地表將改變視覺印象；廠房量體及相關設施物架設時因量體較大，對於鄰近的遊憩據點可能會影響遊客之遊程體驗。  

# (二) 完工營運期間的遊憩影響預測  

# 1. 天然氣電廠及電塔電纜設置後的視覺影響  

預計本廠區開發完成後，廠房設施、煙囪及高壓電塔量體高大，將取代原有寧靜的農田景象，改變現況環境較為顯著；應透過整體規劃設計，可降低人為設施量體所產生的視覺壓力，減輕遊客之遊程體驗負面影響。  

# 2. 遊憩景點之交通可及性  

完工營運後，將恢復既有道路服務水準，對於遊憩可及性及遊客量影響程度不大。  

# 二、 可能影響遊憩點預測與評估  

據遊憩環境調查，對所選取之五處遊憩據點進行評估（詳圖 6.4.2），茲將各據點在施工期與營運期間對遊憩體驗與遊客量方面之影響等級評估結果整理如下：  

# (一) 民雄金桔觀光工廠  

民雄金桔觀光工廠位於最近的計畫電塔西北側約 2 公里，約 4 分鐘車程，是一處以金桔為主題的觀光工廠，當地居民及附近地區遊客為主要遊客來源，自用汽機車為主要交通工具。  

預計施工及營運期間，因地理位置關係，看不到也聽不到計畫區活動情形，不至於影響遊客之遊憩體驗及遊客量；施工車輛往返可能輕微影響本地區交通服務水準，但完工後將恢復原既有交通可及性，整體來說將屬於輕微或無影響的層級。  

# (二) 旺萊山鳳梨文化園區  

旺萊山鳳梨文化園區位於最近的計畫電塔西北側約 1 公里，約 2 分鐘車程，因周邊種植大面積的鳳梨，是一處以生產鳳梨酥為主的觀光工廠，並提供各式鳳梨相關產品供民眾選購。本遊憩據點遊客主要來源為當地居民及外地遊客，自用汽車為主要交通工具。  

本遊憩據點雖離計畫輸電線路不遠，但受周邊建物、植被阻隔及地理位置影響，預測將來施工及營運期間，看不到也聽不到計畫區活動情形，對於本地區遊客之遊憩體驗影響程度並不顯著。施工行為及施工車輛可能影響往返本遊憩據點遊客之行車不便；完工營運後，恢復原道路服務水準，整體來說遊憩品質將是輕微或無影響的層級。  

# (三) 虎頭崁埤  

虎頭崁埤位於計畫廠區東北側約 3.5 公里，約 5-8 分鐘車程，為民雄八景之一，早期因湖水清澈且周邊植被豐富而成為重要的觀光景點，近年來因公共設施缺乏管理，較少遊客前往，多以當地居民為主，汽機車為主要交通工具。受地形及植被影響，預測施工及營運階段，看不到也聽不到計畫輸電線路及廠區內施工活動，對於本地區遊客之遊憩體驗及遊客量無影響；施工車輛往返周邊重要動線，對於交通可及性有輕微的影響，完成營運後即可恢復原本之交通服務水準，整體來說遊憩品質將是輕微或無影響的層級。  

# (四) 卡普秀時空膠囊觀光工廠  

卡普秀時空膠囊觀光工廠位於計畫廠址西南側行車距離約 3 公里處，約 5-7分鐘車程，是一處以保養保健為主題的時空膠囊觀光工廠，規劃許多體驗區及各種產品試用及採購，吸引許多外地遊客前往，自用汽車為主要交通工具。  

本遊憩據點受地理位置影響，施工期間的整地、吊裝等作業活動並不會對本遊憩區造成影響，然施工車輛的往返將增加周邊道路之交通流量，對於交通可及性易造成輕度負面影響，但對於觀光工廠內遊客之遊憩體驗並無影響。未來完工營運後，遊客的視覺及聽覺並不受影響，且恢復周邊道路服務水準，整體來說，對於本遊憩環境影響將是輕微或無影響。

受地理位置及工業區建物阻隔，施工及營運期間，看不到也聽不到廠區內活動情形，不會對本遊憩景點造成影響；然施工車輛的往返將增加周邊道路之交通流量，對於本遊憩據點的交通可及性造成輕度負面影響，完工營運後即恢復原道路服務水準，整體來說，對於本遊憩環境影響將是輕微或無影響。

# 7.4 社會經濟  

# 7.4.1  土地使用  

本計畫廠址位於嘉惠電廠既有廠址位置南側，預定廠址用地多屬台糖公司農牧用地，目前為農種果園及闊葉林種植。本計畫面積約 14 公頃，區分為主要開發區及綠帶，加上廠區與附近聚落已有充份緩衝空間，可減少環境直接衝擊。另由於電廠設立，可因電廠建設因素，帶動附近建設發展，提高就業機會，對嘉義地區就業環境有正面助益。  

# 7.4.2 社會環境  

電廠興建須投入大量人力、財力及物力，非但創造所在區域之就業機會，更可促進地方經濟活動，有助於繁榮地方經濟，造福鄉里。以下針對本計畫所提供電力以外社會效益，列項說明。  

# 一、 國內資源利用程度  

本計畫土木、建築工程數量龐大，建築材料將以國內供應為主。此類材料，如水泥、砂石、鋼筋及五金等絕大部分將自國內採購。機械、電氣等設備採購金額亦極巨大，開發單位亦將在考慮品質能合乎規範要求下，儘可能在國內採購，以增進國內資源使用與經濟成長。  

# 二、 提供地方就業機會  

施工期間所須人力大致包含技術性機電裝配技師、建築技工及一般工人等。其中技術性技師主要來自開發單位及其他工程單位，而非技術性工人可優先雇用當地人力。所有參與施工的人員約可分為以下 3 類：  

(一) 開發單位聘僱人員，即開發單位的工程師群及其工程顧問等。  
(二) 合約工，即承包各項工程之包商所轄工作人員。  
(三) 臨時工，即所雇用之臨時人員。  

另外，在施工及運轉期間，因施工人員、運轉人員及眷屬移入，將促使電廠附近醫療、租屋、餐飲業、娛樂業、交通運輸及零售販賣等行業發展，間接增加就業機會。  

# 三、 對地方工業發展之貢獻  

由於本計畫執行對地區的電力可提供更進一步的保障，相對亦提升大型工業選擇於附近地區投資設廠意願，除提升當地工業規模及水準，亦有助於振興當地產業。  

# 四、 其他  

電廠興建將引進員工及廠外從事商業及服務的工作人員，而人員增加所造成大量日常消費，及在工程進行階段有大量物料進出，均將使區域內地方稅收增加，充裕地方政府的財政。  

# 五、 公共設施  

施工及營運期間對於公共設施需求以醫療體系為首要設施，本計畫將利用電廠附近地區醫院或診所作為緊急意外事件救助之處。此外，施工期間需有電力、自來水、污水處理及垃圾貯存等設備，其中污水將自設套裝式處理設備或流動廁所，由於在施工階段需求量不大，故對嘉義縣公共設施需求影響應屬輕微。  

# 7.4.3 經濟環境  

# 一、 施工階段  

廠址施工期間，對經濟活動的影響為創造營造業就業機會，同時增加地方政府營建稅收，另需依法繳納空氣污染防制費用，供政府執行空氣污染防制措施使用。營建人員因日常生活所需而在廠址附近消費，可增加當地商業收入及地方政府的營業稅收，故對廠址鄰近區域經濟結構具有正面影響。  

# 二、 營運階段  

營運期間電廠運轉期間運轉人員及眷屬移入，將促使電廠附近租屋、餐飲業、娛樂業、交通運輸及零售販賣等行業發展，間接增加當地經濟發展，具有正面影響。

本計畫施工階段對鄰近道路系統造成之交通衝擊，包括施工人員通勤旅次、工程材料或填土運送等施工車輛對於交通之影響，本計畫後續將根據上述評估施工期間尖峰小時衍生車旅次，評估施工目標年（民國 114 年至117 年）對周邊交通環境影響。相關說明如下：

# 2. 衍生車旅次分析  

(1)員工  

有關員工之運具使用比例係依據中華民國交通部出版之「民眾日常使用運具狀況調查－摘要分析」，嘉義縣民眾運具使用比例為汽車 $32.8\%$ 、機車 $52.0\%$ 、自行車 $2.8\%$ 、大眾運輸 $8.2\%$ 及步行 $4.2\%$ 。考量本計畫基地位置較偏遠，周邊 500 公尺範圍內無大眾運輸系統，且施工人員難以步行或騎乘自行車方式通勤，本計畫採保守估計，將大眾運輸、自行車及步行之運具移轉至私人運具，依據比例調整汽、機車運具使用情形。經調整後，員工運具使用比例為汽車 $38.7\%$ 及機車 $61.3\%$ ，乘載率以 1.0人/車估算。

# (三) 施工階段交通維持計畫初步規劃  

1. 施工區出入口處選派專人，指揮施工車輛進出，提醒車輛駕駛注意行車，維護施工安全。  
2. 施工期間若必須佔用車道，除依相關規定向主管單位提出申請外，對於佔用車道之交通管制，標誌、號誌、警示燈等之佈設及規劃設計，應符合交通部編審「交通工程手冊」之作業標準，以確保交通順暢及行車安全。  
3. 施工期間所有材料機具，均需放置於工區內，不得停放堆置於進出道路二側。  
4. 進出動線道路應經常檢視路面狀況，如有破損應立即修復以維道路品質與交通安全。施工區及鄰近道路禁止路邊停車，經常檢查並保持施工區及道路之施工標誌、燈號之清潔及正常運作。

# 7.6 文化資產  

經過資料查核和現地調查，本計畫周邊 500 公尺內已知文化資產有「大丘園遺址」考古遺址類1 件。本計畫基地位於平地與丘陵交接帶，經過多年農耕行為，且有種植稻作，下挖深度推測在現地表下 20-40 公分左右，本次調查時觀察山麓露出地層斷面，並無發現地下埋藏文化現象。鳳梨田種植時有發現使用肥料，顯見當地土壤不甚肥沃，可能不適合史前人群農耕行為（穀類）使用，因此歷年調查皆無發現史前遺跡或遺物。根據調查結果，本計畫基地下挖過地表下 60 公分以下有遇到地下埋藏文化的可能，為避免工程進行時突遇埋藏較深的文化層，施工階段將執行施工監看作業，以避免破壞可能存在的地下埋藏文化。  

無形文化資產方面，調查範圍有內松子腳上升宮、太和無極宮和正法宮等，供奉皆為可能有民俗活動紀錄之神明，本次調查雖無觀察到舉辦大型民俗活動如繞境、會香之記錄，但可能是近兩年疫情影響活動進行，若將來有舉辦上述活動，則會受到工程施工動線等間接影響。因此，建議工程進行時，注意里辦公室等活動公告，或注意沿路是否出現繞境動線指示的閤境平安旗，避免影響當地民眾參與活動的文化權。  

以文化資產保存維護之精神而言，各項營建工程或開發行為遭遇文化資產時，原則上應以迴避為優先考量，若確無法迴避則應提出處理方式建議送請相關主管機關審查。
